Sphinx-4 Hello World! Demo

This is a simple command line program that shows how to write a simple Sphinx-4 application that recognizes connected digits, using a BNF-style grammar. When you run the demo, the following will appear:


run:
[java] Say any digit(s): e.g. "two oh oh four", "three six five".
[java] Start speaking. Press Ctrl-C or say 'good bye' to quit.

It expects you to say a sequence of digits. When you stop, the recognition result will be shown, and you will be prompted to say another sequence of digits. When you want to quit the demo, simply say "good bye".

Running

First, make sure that all the Sphinx-4 classes are built. Go to the top level directory (sphinx4), and type:

sphinx4> ant all

Then make sure that you have JSAPI setup correctly.

To run this demo, type:

sphinx4> cd demo/sphinx/helloworld
sphinx4/demo/sphinx/helloworld> ant run
