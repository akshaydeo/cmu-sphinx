Sphinx-4 Zip City Demo

This is a Java WebStart application that will demonstrate digit recognition

Building

Check if the bin directory already has the ZipCity.jar file. If not, you need to build the demo to create that JAR. To build this demo, merely type the following:

    ant -find demo.xml

Doing so will create bin/ZipCity.jar under the top level directory. You can run the demo using java -jar (see "Running" below).
