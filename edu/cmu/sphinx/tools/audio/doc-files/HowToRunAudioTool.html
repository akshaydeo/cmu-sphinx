Sphinx-4 AudioTool

AudioTool is a visual tool that records and displays the waveform and spectrogram of an audio signal. To run this AudioTool, first make sure that all sources are built by typing at the top level directory: 

ant

Then, to run the tool, at the top level directory, type: 

java -jar bld/lib/audiotool.jar
